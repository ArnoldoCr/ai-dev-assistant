# 🤖 AI Dev Assistant

Aplicación web que integra IA (Gemini) para asistir a desarrolladores en tareas comunes como debugging, optimización y explicación de código.

## 🧠 Funcionalidades

- 💬 Interfaz tipo chat
- 🐞 Modo Debug: detecta errores en código
- ⚡ Modo Optimize: mejora código y buenas prácticas
- 📖 Modo Explain: explica código de forma clara
- 🎯 Prompt engineering para respuestas más precisas
- 📋 Copiar respuestas fácilmente

---

## 🛠️ Tecnologías

- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **IA:** Google Gemini (AI Studio)
- **Otros:** Fetch API, CSS básico
